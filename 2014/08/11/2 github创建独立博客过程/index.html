github创建独立博客过程

把独立博客创建过程的经验写下来

步骤：

1.安装Node.js和github

2.安装Hexo并初始化

3.配置博客过程遇到的问题

4.推荐

1.安装Hexo之前必须安装Node.js和github客户端

通过pkg安装包安装，安装完Node后提示信息
Node was installed at

       /usr/local/bin/node

    npm was installed at

       /usr/local/bin/npm

    Make sure that /usr/local/bin is in your $PATH.

在终端执行以下命令
切换到
cd /usr/local/bin/
如果没有管理员权限会报错

Please try running this command again as root/Administrator.

执行以下操作切换到管理员模式
sudo chown -R $USER /usr/local

    zhanhxinyudeMacBook-Air:bin zhangxy$ sudo chown -R $USER /usr/local

    WARNING: Improper use of the sudo command could lead to data loss
    or the deletion of important system files. Please double-check your
    typing when using sudo. Type "man sudo" for more information.

    To proceed, enter your password, or type Ctrl-C to abort.

    Password:

PS：密码是不会显示出来的。

3.配置博客过程遇到的问题

3.1 在config.yml文件中的链接后缀必须是github.io.git否则会报错

3.2 必须使用空格，而不能使用tab键否则会报错
reason: ‘bad indentation of a mapping entry’,

3.3 在deploy过程会让输入github账号和密码
git config —global user.name “Your Name”
git config —global user.email you@example.com

成功会提示
[info] Deploy done: github
3.4 自定义域名问题

在Hexo目录下的source文件夹下需要建立CNAME文件，用windows下地记事本建立，如果是mac就去下载个txt文本文件修改并改名，不能有后缀。CNAME文件里填上自己的域名地址，不能有多余的内容，如只能有
baobaoibaobao.com
